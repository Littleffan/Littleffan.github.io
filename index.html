对象不在的这一周

发表于 2025-07-23 更新于 2025-07-25 分类于 日记

ONE

对象从7月5号正式离职后，想到了可以去其他省份找朋友玩，所以7月15号早晨六点多从浦东机场出发了。我们赶在15号走之前，连续两个晚上都搬东西换房间。所以都很疲惫，出发也在匆忙之中出发了。我困到回来继续睡觉了两个小时，可以说是到家就在床上昏迷了。

晚上下班回来，又连续两三个晚上我自己在收尾，总算是把东西全部挪过来了。搬家这事，经历过两次后，就知道了别看房间了大件衣服箱子都弄走后就没活了。其实剩下的零散小件也是着实不容易收拾，拾掇拾掇一包又一包。

挪到新房间后，要收拾、摆放，采购新的纱窗纱门。各种下水口也要进行再密封。所以又搞了几天。阳台上物件，地板花盆之类的就还是没来得及处理。

一个人呆的时间也理解了对象在家的时候真的不轻松，喂鱼、喂猫、浇花、铲屎、洗衣、做饭、清扫、整理等等，这一套下来还是需要不少时间和精力的。幸而有两个猫陪着，不然我一个人住是真有点害怕。周围的几户都没有人，我们又有一个大露台。对着窗子睡觉的时候，我就告诉自己，猫是最机敏勇敢的。如果连它们在黑夜里都没发现什么的话，就是最安全的。况且奶盖也会时不时的跳到我身边来呼噜。我更加不用去多加担忧了。

TWO

7月24号的飞机回来，我去浦东机场接。飞机晚点一个小时。晚上10点，在去的路上，脑海里不自觉的浮现要是发生车祸怎么办，是不是就没法准时到机场接机了。所以我不断告诉自己，稳一点、慢一点，保持车距能让则让。

但是意外还是发生了，外环高速上我右前方的大货车突然爆胎。第一时间完全没意识到是货车爆胎，只是突然一声巨响，人有点懵。然后才观察到右边一个身位的货车。紧接着就是心脏开始剧烈跳动，本来疲倦感也瞬间消失了，久久不能平静。越是回想越是害怕。

意外来临的时候，真的是一点不给你反应时间，并且发生的瞬间人完全是无知的。所以才叫意外吧。回家的时候凌晨12点多了，上海外环高速上仍然车流不息。在上海这座城市奔波打拼的人，耗尽心血后会不会觉得还是活着就好。

对象玩的最后是一个人报了个旅游团，她说，车上的导游是按照家庭来分组的，基本每组都是1-4个人。只有她是一个人一组。进景区的时候排了一个半小时的队，进去之后也并没有好一些，狭窄的沿河路上仍然挤满了人，大家只能缓慢的向前移动。逛完一个景点，前往下一个景点，因为间隔较远只能乘坐大巴车，这又要开始排队，至少半小时起步。在这偌大的景区内，只能这般排了又排。再好的心情也会被疲惫掩盖。

利用Typora和PicGo和阿里云OSS搭建博客图床

发表于 2025-07-18 分类于 Hexo

我通常使用Typora来进行博客写作，但是图片是以“外链”的形式存在的。如果Typora里面插入的图片的源文件被移动或者删除了或者博客发布了，博客里面对应的图片也无法显示出来（因为这个图片只是一个你硬盘里面的文件路径）。

在我们往typora里插入图片的时候，PicGO会自动上传图片到图床，并返回一个图片的链接。即便是本地的那张图片移动or删除，也不会影响图床里的这张图片markdown本地文件里面的图片也就不会失效了！

具体详细步骤参考Typora+PicGo+阿里云OSS搭建博客图床（超详细），作者：慕雪华年

1.安装Typora和[PicGo](Releases · Molunerfinn/PicGo)

2.购买和配置阿里云OSS

创建Bucket，Bucket名字不能有大写字母；服务器就近选择；图床选择标准存储；读写权限公共读；同城冗余存储选择关闭；版本控制选择关闭。

找到Bucket-概览-访问域名下的地域节点；

找到账户资料里的AccessKey进去后创建一个AccessKey，在弹出的界面里，记住你的accessKeyId和accessKeySecret；

3.配置Picgo

在图床设置里面选择阿里云OSS，依照以下步骤填写信息

设定Keyld：填写刚刚获得的AccessKeyID

设定KeySecret：填写AccessKeyIDSecret

设定储存空间名：填写bucket名称这里填写的是bucket名称，不是浏览器里的域名

确认存储区域：填写你的地域节点，注意复制的格式

指定存储路径：其实就是自定义一个文件夹的名字，以/结尾它会自动在你的bucket里面创建一个文件夹，并把图片上传进去.

弄完之后，记得“确定”，并点击“设置为默认图床”！

在Picgo设置里打开时间戳重命名和上传后自动复制URL

4.配置Typora

进入typora主界面，点击左上角的“文件-偏好设置”

选择图像

插入图片时上传图片

上传服务选择PicGo(app)

PicGo路径：找到picgo的安装路径不是安装包的路径！！！！

2024年日记-3月

发表于 2024-03-18 更新于 2025-07-25 分类于 日记

2024年3月18日

不是我故意暗示我自己不喜欢这里，是真的难以适应环境。

周末是怎么都不舒服，感觉呼吸不畅，吸入的氧气怎么也不够，就很累。我们周日去的梅林水库，去的路上太阳光不是很强烈，但是体感比较热并且没有风，毛孔却无法排除汗液，给人又闷又热的感觉。从下梅林地铁B3口出来，没走几步路就已经累了。我和小枝皆如此。

不过到了水库底下，出现了几棵大树，有阴凉也有风了，瞬间整个人活了过来。疲惫感消散。但是开始沿台阶往水坝 上爬，没了风和阴凉不适感又回来了。神奇的是，我们只要吃一颗小西红柿就充满一点活力，像是大力水手吃菠菜一样。一颗一颗的吃着，也就慢慢往山里走去了。

2024年日记-1月

发表于 2024-01-21 更新于 2025-07-25 分类于 日记

2024年1月21日

周五的时候，公司12月份的餐补到账了。充值在给到app里，不能提现，只能通过它内置的一些平台进行消费。

小枝和我决定去看“年会不能停”，比价了一番各平台价格都一样，从给到上下单了，票价43元。电影很好看，讲述的是大鹏演的一个标准件厂钳工阴差阳错被调到总部，误打误撞当上高级经理，又无意间签字裁掉了标准件厂的三百余人，从而意识到问题，再年会上来了一出打工人宣泄。欢声笑语中，也揭露着电影院各个人的伤疤，大家笑着笑着也在年会宣泄中基体沉默。我们真的累了，想要找到生活的意义。我觉得之前都是职场人在网络上吐槽，现在已经在电影层面传播到大众面前，这是进展到一定阶段的表现。

看完我们去莲花山公园散步，唱歌很有状态。

晚上回来吃完饭就打起了王者荣耀，今天晚上很顺利，我玩辅助，小枝玩中单或者射手，我们晚上一直胜利，并没有搞心态，开心的结束有戏。

周日醒来的时候是10点了，我从昨天开始就一直馋甜食，所以小枝从沃尔玛给我买了400g的提拉米苏29.9元，我准备了午饭，西红柿炒鸡蛋，白菜虾牛肉丸蟹棒汤，昨晚的微波炉烤茄子，配上车厘子。吃完饭我刷了会B站，了解Docker是什么，html-css-js是什么，NAS的小白讲解。

然后小枝给我剪头发，上一次是2019年了，那一次因为第一次剪，效果不好，最后只好剃成光头。今天拿出时隔三年的套装，剃刀却已经饿死了，冲一小时的电也启动不了。没办法，只好用剪刀了。为了防止碎头发没法清理，我们就在卫生间，我坐在马桶上，小枝站着。历史一小时，主要就是将两边和后面剪短并剪出层次，参考的最近流行的美式前刺发现，最后用发膏一抓，是真不错，虽然只是第二次剪，但是已经和上次是天壤之别。

我爸给我打了电话，问了一下我几号回家，买票了没。然后告诉我柳那边还没打钱。我说没事，等着吧。不过我也是着急，但是已经问了柳两次了，又不好再去催。也只好先等着。

晚上又玩起了王者荣耀，不过今晚没有一直赢，最后的两三把全是输的，不过小枝已经进步很大了，我们在晚上10点40左右停止不玩了。所以我写起了日记，这也是时隔很多年了，上次正式写日记应该是2017年在学校了。现在脑子真实空白，只能记流水账。不过，加油。先记录吧。

2024年1月22日

今天开始降温了，不过是5-14℃，我觉得不冷，一件华夫格单长袖，一件稍微厚点的外套，单层牛仔裤。

不过办公室的人大都穿上了羽绒服。可能这里呆久了都会怕冷吧。从早上开始温度是一点一点的下降，早晨出门的还可以敞怀，到了晚上6点半下班就必须扣好扣子了。我倒是喜欢这样的温度，微凉。却不喜欢一直不下雨，我一直在等雨，从九月份到这里就盼望着。我暂时不知道为啥想要下雨，可能是一尘不变的晴天让人感到厌倦。

昨天晚上失眠了，先是感到盖着被子热，把腿伸出来后又冷了起来。渐渐又出现尿意去上厕所，回来辗转反侧怎么也睡不舒服，脑海里开始不自觉的出现各种事情，什么未来工作地、家人健康、我如何去养老、结婚、明天工作如何开展等等，实在是思绪纷飞，干脆刷会手机，刷着刷着又开始上厕所。也不知道是不是我睡前喝了菊花茶的原因。大概凌晨1点36分的时候，我明天是要上班的，所以强制自己睡觉。接下来就不知道什么睡着了，迷糊中听到外面有母亲大喊大叫的声音，实在困得不行，只在心里想着“这不得大家都去投诉嘛”就又睡着了。可能是快天亮了，猫又开始哐当哐当，我实在困又生气为啥会被吵醒，只好起来吓唬它们一下。

这一夜睡得实在不行，去公司了，精神打不起来。上午也难以集中精神工作，就摸摸鱼了，学习了下HTML，了解了它的基本语法，这样下次再F12的时候希望可以能看明白一点。下午才开始工作，然后去找领导做周汇报，他终于是抽出了时间，推迟了两周后对我进行了试用期的第二次考核，问题是对上次我回答不好的重复，这次我当然已经烂熟于心。马上两个月了，我还是没有什么正式工作，只做做新项目的调研，然后熟悉公司流程，采购原材料进行样品制作。有点难以琢磨。之前说着要给我难度高的工作，可是也是两周过去了还是没想到具体的工作。等着看看吧。

晚上回家后，吃完饭，又和小枝开始了王者荣耀，到了10点40左右吧。有输有赢。我做好辅助，提高小枝的游戏体验就行。

2024年1月23日

昨天刚说到下雨，今早就下起了雨。早晨去阳台拿袜子没带眼镜，模糊地看见对面楼顶潮乎乎一片，心里想应该不会是下雨，可能是雾水吧。

没想到出门，还未感觉到雨丝，却见来往的人举着雨伞。还好，毛毛细雨。

今天的温度是4-8℃，是我来了之后最低的温度，也应该是今年冬天最低的温度了。地铁上已经基本都是穿羽绒服的人。

我还是昨天的外套，只不过是换了件贴身长袖和一件厚点的毛衣。出了地铁站，雨稍稍有点大，只好快步往公司走去。

对于早饭，我没啥欲望。之前的工作，总是在地铁站的全家买面包或者饭团。现在因为出口处没有便利店，都是小摊小贩，我基本上每天都是买的麦当劳早餐，用上每个月的早餐券后平均一份早餐是12元（一大杯豆浆+一猪柳蛋/卷+油条/土豆饼），目前为止应该吃了一个月了，期间只是偶尔买个小摊贩的茶叶蛋八宝粥和红薯。觉得没啥意思，只是填饱肚子而已，其实并不享受。

下午是去厂区开始做样品了，到了才发现我要去找的那个干了5年的技师正在走离职流程，明天就不来了。我说你选择这个时间点走，是年终奖都不要了么，再两周就春节了。技师和我说，领导给他绩效打了最低的那一档。我就明白了。公司从把研发大部门打散也就差不多一年时间，他虽然5年司龄但是在这个部门也就一年。其实很不理解为什么我昨天去找领导说接下来会去找这个技师一起做样品的时候，领导啥也不和我说。

今天听小枝说，她今天认识了一个新同事，和我们同岁。刚结婚一年，所有家人都在老家，只身一人在这个城市工作，时常周末的时候坐飞机往返。初始听到这样的选择，觉得她真的勇敢。没想到她是2023年入手的房子，时间点正好是高位下降的过程中，背负几百万的贷款。那来这里工作赚钱的速度是会快一些。只不过在这样的压力下，还要新人异地，挑战真不是一点半点。不过又能怎么办。曾经2019年的时候，我们也是有强烈的买房欲望。那个时候觉得只有买了房子才能结婚，才能获得安全感。也看了一段时间的房子。身上现金不多也是一个原因，导致搁置了。再后来伴随这认知的成长。房子所能提供安全感这个观念渐渐变成一个伪命题。现在这个房价是在掉，很多人为此付出惨痛代价，但是谁又能预料以后会不会再涨回来。我们大部分人并没有判断房价走势的能力，却要把安全感抑或幸福感或者人生的奋斗目标寄托在社会追捧的物质上，是很容易求而不得的。

在我写今天的这个后半段的时候，因为我的一句话，导致我们又吵架了。我不玩游戏了之后就去洗碗晾衣服去了，但是小枝还在玩，并且一直输，网络还总是卡。她在和我抱怨，心态接近奔溃的时候，我就说：不要玩了吧，不要在意他们呀，输赢无所的呀balabala。就是这些话，让小枝觉得我是看不起她。哭了很久。我怎么多的话，都像是在狡辩。最后我关了灯，去洗手间坐着，不发出声音。

2024年1月24日

虽然昨晚睡得晚，早上还是7:10闹钟响的时候就起来把昨晚洗好的糯米、黄豆和桂圆放进破壁机了。

做五谷浆需要30分钟，这机器的噪声以前倒不觉得，到了这个比较小的厨房，加上厨房桌面也不是很厚实，所以格外的吵。这30分钟我战战兢兢，甚至在心里已经演绎出一套邻居来敲门的剧情了。我会试着义正言辞地说：你别投诉我啊，我用正常电器正常地做早饭，应该去投诉房屋质量差隔音太差。好了之后就倒了一碗在厨房冷着，小枝起床后正好喝。小枝也说这机器太吵，导致她比以往少睡了半小时，下次还是不要早上打豆浆了。

今天还是不想上班，也不想去厂区。所以就研究了昨天布置给我的医工合作的事情。万没有想到，这个产品竟然公司十年前就做过了，还恰巧就是我旁边的TC参与的，他甚至测过竞品。他也无法理解为什么又要重新做。我说这是重复造轮子罢了。当初的那些人大都不在了，领导们并不关注真正在做什么吧，我们和我们所作的事情只不过是他们糊弄上级争取利益的工具罢了。

2024年1月26日

这一周都没有睡好，就算早也得12点这样才能睡着。领导在我入职第一天说，公司虽然上下班弹性一个小时，但是我们部门最好8点半到，要是晚一点也要9点之前，只有特殊情况才能在9点在9点半之间。我很不能理解，那这个弹性有什么意义呢。

2024年1月29日

又是一周，不想上班，不想工作。小红书说我们每天是拿8小时的时间换4小时的休息。不过我好像是每天接近10小时的时间，换来4小时休息，睡觉都不足8小时。

上周五的晚上是部门聚餐，领导提前一周告诉我要预留时间出来。但是一直没和我说地点及时间，直到下午六点钟了，企信上同事的状态陆续从离线变成下线了，还是没人来告诉我。我觉得他们肯定是忘记我了，但是我要是不问肯定会让领导面子上挂不住。所以还是假装不清楚计划的样子，去问是不是今晚聚餐取消了。领导显示了好一会正在输入中，给我回了句：抱歉，忘记告诉你地址了。随后给我发了定位让我赶快过去。我就知道是这样。到了地方，发现包间里两张桌子，早就坐满了，估计是等了我有29分钟吧。我也没啥不好意思的，又不是我的原因，记得上周领导和我说，我暂时还不方便透露太多信息，那我就安安静静的吃，也省去了打招呼的麻烦了。

周六决定去把车子开出去溜溜，放在公司底下也有三周了，只是上周的时候我下班的时间在停车场开了10分钟。担心10分钟电池充不了多少电，决定这周末去远一点的地方。本来想去顺德或者江门，但是由于昨晚小枝在看小说刷小红书我在看繁花，凌晨一两点睡得，周六中午才起床，做饭吃饭后也就到下午2点了，搜了一下无论去哪个城市都要2个小时的车程，去我公司还得40分钟。好吧哪也去不了，小枝临时小红书搜了个红树林公园，15公里显示30分钟车程。开了20分钟后，导航显示还有接近30分钟时间，小枝说：你看，这不又要一个小时了么，多好。哈哈哈，深圳这交通，有的时候堵的反而会让你感觉不错。红树林公园在海边，没有沙滩，是一片滩涂，滩涂上长着一片蜡烛果，裸露的石头上和观景台的石柱上都是附着的贝壳。我不是很舒服，感觉自己飘飘的，风景都看不进脑袋里，加上这里风大随后又飘起了小雨，我们便返程了。正好公司附近有个万象天地，便决定过去逛逛。这里的店是真的挺大的，1层全是大牌化妆品店或者香奈儿这种奢侈品店，我们要去找点吃的，只能去到5楼的大时代。这里还有很大的华为和小米体验店，不愧是在科技园开的商场。

上周去莲花山公园的时候，说是想要来这草地上坐着看书，每次都想要有个野餐垫不如立马就下单。结果这周野餐垫到了，天气却不好，阴冷。所以作罢，周日在家又是中午起床的，做饭吃饭刷碗，然后玩游戏，我又看繁花，不知不觉就到五六点了。想着是不是可以写下日记，但终是难以静下心。

2024年1月31日

早上起来洗漱的时候发现厨房瓷砖地板湿湿的，我原本以为是昨晚可能洗碗什么的弄在地上没干。到公司听同事说，这就是深圳的回南天，他们都开始用起了抽湿机。提醒我，最近不要打开窗户，空调有抽湿功能就打开。

昨天晚上12点左右，一个老太太在门口打电话，声音一会儿高起来一会儿低下去。可能得有五分钟，我已经在忍耐的边缘了，在犹豫要不要出去警告一下。正好这时小枝也被吵醒，顺势从床上坐起来了。我就直接开门去了，这人也是鸡贼。我们住的就是公寓的那种布局，长长的走道两边分布着十几户。这个老太太明显不是我这一边的，她打电话不在自己房间或者门口打，非要跑来我们这边的窗口。我说：你打电话你就回你自己房间打，不要吵别人睡觉。她一声不响的就走了。

说起来也搞笑，下午同事让我下去领苹果，我以为是周六我来开车的时候在大堂看到的一箱箱苹果。当时和小枝还在开玩笑，公司怎么这么抠，年货就发一箱苹果并且还是领导特定封面红包的人才能领。没想到，下去却领了2个苹果和2个橘子。哈哈哈哈笑死我。这怎么好意思发的！我自己晚上在淘宝买了点年货礼盒给家里寄回去。
